
[49. Group Anagrams](https://leetcode.com/problems/group-anagrams/)

код решения

``` python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        res = {}
        for el in strs:
            letters_cnt = [0] * 26
            for letter in el:
                letters_cnt[ord(letter) - ord("a")] += 1
            key = tuple(letters_cnt)
            if key not in res:
                res[key] = []
            res[key].append(el)

        return res.values()
```
